# Backend Skripsi v2

Ini adalah backend untuk project akhir. Dibangun di atas Node.js platform dan menggunakan Nestjs framework untuk mempermudah bootstraping app. Project ini menggunakan mongodb sebagai database nya.

## Installation

```bash
$ npm install
```

or

```bash
$ yarn install
```

## Install mongodb

install mongodb (skip if you run in docker) [download here](https://www.mongodb.com/try/download/community)

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# run in container
$ docker compose up
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
